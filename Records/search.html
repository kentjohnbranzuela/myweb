Search Functionality

Ang "Search" ay isang mahalagang bahagi ng system na nagbibigay-daan sa mabilis na paghahanap ng datos.

Purpose

Layunin nitong mapadali ang paghanap ng tiyak na impormasyon sa database upang mapabuti ang workflow.

Advantages

  • Mabilisang pagkuha ng datos
  • Mas epektibong data management
  • Pinasimpleng proseso ng paghahanap

Disadvantages

  • Maaaring hindi agad mahanap kung hindi tama ang keyword
  • Posibleng bumagal kung sobrang dami ng records
  • Kailangang i-optimize para sa mas malaking database
